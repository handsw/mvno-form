# 📱 아정당알뜰폰 현대화 가입신청서 (AJD MVNO Form)

본 프로젝트는 기존의 복잡한 A4 레이아웃 가입신청서 양식을 모던하고 세련된 사용자 경험(UX)을 갖춘 대화형 웹 애플리케이션으로 업그레이드한 프로젝트입니다.

---

## ✨ 핵심 현대화 기능

1. **대화형 폼 위저드 (Wizard Mode)**:
   - 모바일 및 웹 브라우저 환경에 최적화하여 5개 단계별로 필요한 정보를 순차적으로 입력받습니다.
   - 선택 카드 방식의 라디오 버턴 인터페이스로 터치 조작이 편리합니다.

2. **A4 실시간 미리보기 (Print Preview Mode)**:
   - 상단 토글을 통해 인쇄될 실제 A4 양식을 화면에서 실시간으로 확인할 수 있습니다.
   - 위저드에 입력된 모든 데이터가 미리보기 양식에 실시간으로 동기화(양방향 바인딩)됩니다.

3. **디지털 서명 패드 (Digital Signature Pad)**:
   - HTML5 Canvas 기술을 내장하여 모바일 터치나 마우스 드래그로 직접 서명할 수 있습니다.
   - 서명된 이미지는 인쇄 양식의 모든 서명란(1페이지 및 3페이지)에 고화질로 자동 삽입됩니다.

4. **스마트 벨리데이션 및 자동 입력**:
   - 오늘 날짜(년, 월, 일)가 서명란에 자동 기입됩니다.
   - 출력 전 필수 기입 사항 누락 시, 누락된 필드로 부드럽게 스크롤 및 포커싱 이동되어 입력 수정을 유도합니다.
   - 인쇄 시에는 상단 메뉴 및 화면 UI 요소들이 자동으로 배제되어 깨끗한 A4 문서만 출력됩니다.

## 🌐 개인 아이디(`handsw`) 없는 배포 및 도메인 설정 방법

개인 깃허브 아이디가 URL에 드러나지 않도록 배포하는 세 가지 최적의 무료/유료 솔루션입니다.

### 옵션 1: Vercel 배포 (무료, 가장 추천)
Vercel은 깃허브 저장소를 연동하여 클릭 몇 번으로 초고속 정적 사이트를 무료 배포할 수 있는 플랫폼입니다.
- **제공 주소**: `https://[원하는프로젝트명].vercel.app` (예: `https://ajd-mvno.vercel.app`)
- **방법**:
  1. [Vercel](https://vercel.com)에 가입하고 깃허브 계정을 연동합니다.
  2. **Add New > Project**를 누르고 복사된 `mvno-form` 저장소를 선택하여 Import합니다.
  3. **Framework Preset**을 `Other`로 두고 **Deploy** 버튼을 누릅니다.
  4. 단 10초 만에 배포가 완료되며, 개인 아이디가 보이지 않는 주소로 서비스가 제공됩니다.

### 옵션 2: Cloudflare Pages 배포 (무료)
글로벌 CDN 서비스인 Cloudflare의 무료 정적 웹 호스팅 서비스입니다.
- **제공 주소**: `https://[프로젝트명].pages.dev` (예: `https://ajd-mvno.pages.dev`)
- **방법**:
  1. [Cloudflare 대시보드](https://dash.cloudflare.com)에 로그인합니다.
  2. **Workers & Pages > Create > Pages > Connect to Git**을 차례로 누릅니다.
  3. 깃허브의 `mvno-form` 저장소를 선택하고 배포 설정을 진행합니다. (Build Command 없음, Directory는 `./`)
  4. **Save and Deploy**를 누르면 배포 주소가 활성화됩니다.

### 옵션 3: 개인 도메인 구매 후 깃허브/Vercel 연결 (유료, 전문적 브랜드 구축)
가장 추천하는 형태는 `www.ajdmobile-apply.co.kr`와 같이 가비아, 후이즈 등에서 도메인을 만 원 안팎으로 구매하여 연결하는 방법입니다.

#### 깃허브 페이지(GitHub Pages) 도메인 설정법:
1. 도메인 판매처에서 CNAME 레코드에 `handsw.github.io`를 가리키도록 설정합니다.
2. `mvno-form` 저장소 루트 디렉토리에 **`CNAME`**이라는 대문자 파일을 새로 생성하고, 내부에 구매한 도메인 명칭(예: `ajdmobile-apply.co.kr`)을 기입하여 푸시합니다.
3. 깃허브 레포 Settings > Pages 메뉴의 **Custom domain** 칸에 도메인을 입력하고 저장합니다.

#### Vercel / Cloudflare 연결법:
- 배포 플랫폼 설정(Settings > Domains) 화면에 도메인을 입력하면 대시보드에서 도메인 판매처에 추가해야 할 CNAME 값과 IP(A 레코드) 정보를 안내해 줍니다. 그대로 도메인 관리 대시보드에 적용하면 즉시 연결됩니다.

## 🛠️ 로컬 개발 및 실행

프로젝트 폴더 내에는 별도의 빌드 도구 설치가 필요하지 않습니다. 아래 명령어로 즉시 로컬 서버를 구동해 볼 수 있습니다.

```bash
# Python을 이용한 간이 서버 구동
python -m http.server 8080
```
이후 브라우저에서 `http://localhost:8080`으로 접속하면 작성 화면을 확인할 수 있습니다.
